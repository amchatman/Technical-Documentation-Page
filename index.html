Introduction

CSS (Cascading Style Sheets) is the code that styles web content. The stylesheet language CSS is used to describe how an HTML or XML page will appear.

CSS is not a programming language, just like HTML. It's also not a markup language. HTML elements can be styled selectively using CSS.

Prerequisties

This guide assume you know the following

  • A basic understanding of HpyerText Markup Language (HTML)
  • A basic undertading of the World wide web.
  • Know how to link a stylesheet from your HTML page.
CSS basics

To get started with CSS open a HTML file and link it your CSS stylesheet.

p{ color:green};

The code above will change the HTML element p to the color green.
